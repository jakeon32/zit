📝 2. Prompt Structure

동일 스타일을 4가지 프롬프트 패턴으로 비교. 미디엄퍼스트 vs 주체퍼스트 vs 묘사만 vs 아티스트레퍼런스.

Top 5 Rules

  1. Style First — 프롬프트 맨 앞에 스타일/미디엄 배치
  2. Natural Language — 태그 나열 대신 자연어 문장형 (Qwen3 최적화)
  3. Affirmative Only — 네거티브 프롬프트 없음. 원하는 것만 명시
  4. 20~30 Words Sweet Spot — 너무 짧으면 모델 기본값, 60단어+면 스타일 shift
  5. Clean Background = Style Consistency — 배경 제거하면 그림체 통일도 상승
