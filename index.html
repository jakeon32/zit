Style Prompting Tips
  • 스타일을 맨 앞에 — "Watercolor painting of..." 형태가 가장 안정적
  • 자연어 문장형 — Qwen3는 태그 나열보다 문장형에 더 잘 반응
  • Tier 1 스타일(사진, 디지털아트, 3D, 애니메, 유화)은 키워드만으로 충분
  • Tier 3 스타일(과슈 등)은 반드시 스타일명 포함 필요

2. Prompt Structure Comparison

★★★★★

"[캐릭터], loose brushstrokes..."
★★★★ 스타일 shift

"In the style of Agnes Cecile..."

Top 5 Prompting Rules for ZIT
  • 1. Style First — 프롬프트 맨 앞에 스타일/미디엄 배치
  • 2. Natural Language — 태그 나열 대신 자연어 문장형 사용 (Qwen3 최적화)
  • 3. Affirmative Only — 네거티브 없음. 원하는 것만 명시
  • 4. 20~30 Words Sweet Spot — 너무 짧으면 모델 기본값, 60단어+면 스타일 shift
  • 5. Environment = Automatic Context — 환경 묘사하면 조명+분위기+소품이 함께 생성
